# FinSight AI Copilot

FinSight AI Copilot, a Python-based financial analysis tool, enables users to query over 51,000 financial records in plain English without requiring SQL skills. The tool was created for Traxovian Inc., a fictional SaaS company, and includes statistics for 6 years (2020-2025) on sales, expenses, payroll, and budget. The automated Python data pipeline cleans and stores raw CSV files in a SQLite database. Groq AI (LLaMA 3.3 70B) converts natural language questions into SQL queries and provides expert explanations. The dashboard, created with Streamlit and Plotly, offers KPI cards, revenue trend charts, budget vs. real analysis, and a live AI conversation interaction. Developed an AI cost governance module tracking $38,000 in spend across 84,000 sessions, and 28 models. Implemented budget alerts (50–100%), department-level heatmaps for overspend, and a performance classification system to evaluate session-level ROI, providing leadership with real-time visibility into AI efficiency across 8 departments.

## Required Tools
* Python:pandas
* SQLAlchemy
* SQLite
* Groq AI (LLaMA 3.3 70B)
* Streamlit
* Plotly
* Python-dotenv

## Tools and Technologies learnt/used in making this project
* Python scripts that automatically ingest, clean & normalize raw CSV files 
* SQLite relational database storing all 51,000+ records across 6 tables 
* Groq AI running LLaMA 3.3 70B-Converts plain English questions into SQL queries automatically, & rns the query on the database & returns a professional explanation 
* Built with Streamlit such as a full interactive web app in pure Python
